## ⚡ Part 1: Advanced Iteration Tools

### `range()` — The Sequence Generator

`range()` generates a sequence of numbers efficiently, without creating a list in memory. Three forms:

```python
for i in range(5):            # 0, 1, 2, 3, 4
    print(i)

for i in range(3, 8):         # 3, 4, 5, 6, 7
    print(i)

for i in range(0, 11, 2):     # 0, 2, 4, 6, 8, 10
    print(i)

for i in range(10, 0, -1):    # Countdown: 10, 9, 8, ... 1
    print(i)
```

### `enumerate()` — Index + Element

Adds an automatic counter when iterating. Returns `(index, element)` pairs. Eliminates the clunky `range(len(list))` pattern:

```python
amulets = ["Eye of the Universe", "Kiss of the Seraph", "Iron Fist"]

# Elegant
for index, amulet in enumerate(amulets, start=1):
    print(f"{index}. {amulet}")
```

### `zip()` — Iterate Multiple Sequences in Parallel

Joins multiple iterables element by element. Stops at the shortest sequence:

```python
names = ["Yamil", "Ana", "Carlos"]
ages = [25, 28, 30]

for name, age in zip(names, ages):
    print(f"{name} is {age} years old")
```

## ⚡ Part 2: Flow Control Within Loops

### `break` — Exit the Loop

Terminates the loop immediately. Like a samurai's decisive cut — when the target is found, the search ends:

```python
amulets = ["Eye of the Universe", "Kiss of the Seraph", "Iron Fist"]

for amulet in amulets:
    if amulet == "Kiss of the Seraph":
        print(f"✅ Found: {amulet}")
        break
    print(f"Searching... {amulet}")
```

### `continue` — Skip to Next Iteration

Skips the rest of the current iteration and jumps to the next one:

```python
for i in range(10):
    if i % 2 != 0:   # If odd
        continue      # Skip
    print(f"Even: {i}")
# Prints: 0, 2, 4, 6, 8
```

### `else` on Loops — Execute If No Break Occurred

`else` after a loop runs **only if the loop completed normally** (without `break`). Useful for confirming that a search found nothing:

```python
numbers = [1, 3, 5, 7, 9]
target = 6

for num in numbers:
    if num == target:
        print(f"✅ Found: {num}")
        break
else:
    print(f"❌ {target} is not in the list")
# ❌ 6 is not in the list
```

## 🎯 Part 3: Nested Loops

A loop inside another loop — useful for matrices, combinations, and multidimensional structures:

```python
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i * j}")
    print("---")
```

## 🧘 Part 4: Comprehensions — Compact Loops

A concise way to create lists, sets, and dicts using loops in a single line. This is where Python's elegance truly shines.

**List comprehension:**

```python
# Traditional
squares = []
for i in range(10):
    squares.append(i ** 2)

# Comprehension (more elegant)
squares = [i ** 2 for i in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# With condition
even_squares = [i ** 2 for i in range(10) if i % 2 == 0]
# [0, 4, 16, 36, 64]
```

**Dict comprehension:**

```python
squares_dict = {i: i ** 2 for i in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
```

**Set comprehension:**

```python
squares_set = {i ** 2 for i in range(-3, 4)}
# {0, 1, 4, 9} — no duplicates from negatives
```

## 🔑 Summary

| Tool | Purpose |
|------|---------|
| `range()` | Generate numeric sequences |
| `enumerate()` | Index + element pairs |
| `zip()` | Combine iterables in parallel |
| `break` | Exit loop immediately |
| `continue` | Skip to next iteration |
| `else` on loops | Confirm no break occurred |
| Comprehensions | Compact, elegant loop expressions |

These tools transform simple loops into elegant, efficient code. They are not mandatory, but they make your code more Pythonic — and the Pythonic way is always the way of clarity.
