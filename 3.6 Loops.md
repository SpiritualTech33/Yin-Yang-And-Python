## FOR: The Definite Loop

`for` iterates over a sequence (list, tuple, string, range) and executes a block of code once per element. You know how many times it will run — as many as there are elements. Ideal for traversing collections.

```python
traditions = ["Buddhism", "Taoism", "Hermeticism", "Stoicism"]

for tradition in traditions:
    print(tradition)

# If you add a fifth tradition tomorrow, the code already works — no changes needed
```

What happens internally: Python takes the first element, assigns it to the variable, executes the indented block, repeats with the next element, and terminates when there are no more.

```python
# Mantra 108 times
for number in range(108):
    print(f"Om Mani Padme Hum — Repetition {number + 1}")

# If you don't need the number
for _ in range(108):
    print("Om")
# _ is Python convention for "variable I won't use"

# Iterating over a dictionary
chakras = {"Root": "Red", "Heart": "Green", "Crown": "Violet"}

for name, color in chakras.items():
    print(f"{name} → {color}")

# Filtering elements
batting_averages = [0.250, 0.350, 0.180, 0.420, 0.295]
elite = []

for avg in batting_averages:
    if avg >= 0.300:
        elite.append(avg)
# elite → [0.35, 0.42]
```

**Use `for` when** you have a collection to traverse, you know exactly how many times to repeat (using `range()`), or you want to process each element of something.

## WHILE: The Indefinite Loop

`while` repeats as long as a condition is `True`. You do not know how many times it will run. It stops when the condition becomes `False`.

```python
counter = 0
goal = 10

while counter < goal:
    print(f"Breath {counter + 1}")
    counter += 1  # CRUCIAL: increment or it never ends

print("Breathing cycle complete")
```

The `counter += 1` is essential. Without it, the counter would always be 0 — infinite loop.

```python
# Validating user input
age = int(input("Enter your age: "))

while age < 0 or age > 120:
    print("⚠️ Invalid age. Try again.")
    age = int(input("Enter your age: "))

print(f"✅ Valid age: {age} years")

# Intentional infinite loop with break
while True:
    command = input("Enter command (or 'quit' to exit): ")

    if command == "quit":
        print("👋 Goodbye")
        break
    else:
        print(f"Executing: {command}")
```

**Use `while` when** you do not know how many iterations you need, you want to wait until something happens, you need to validate input repeatedly, or the exit condition is dynamic.

---

## The General Rule

**`for`** → when you know the sequence or the number. **`while`** → when you depend on a dynamic condition.

Use `break` and `continue` wisely — if you need many of them, your logic likely needs restructuring. Use them for clear early exits, not for spaghetti code.
