## 🔒 What Are Reserved Words?

**Reserved words (keywords)** are words that carry special meaning for Python — you **cannot use them as names** for variables, functions, or classes. They are the **fundamental vocabulary** of the language.

```python
# Python has 35 reserved words
False, None, True, and, as, assert, async, await, break,
class, continue, def, del, elif, else, except, finally,
for, from, global, if, import, in, is, lambda, nonlocal,
not, or, pass, raise, return, try, while, with, yield
```

### Why Do They Exist?

Imagine Python allowed you to write `if = 10`. Now `if` is a variable. Later you write `if age > 18:` — how does Python know whether `if` is your variable or a conditional? **Total ambiguity.** The interpreter cannot parse the language. Reserved words eliminate this problem by establishing **definitions that are absolute**: `if` is always a conditional, `for` is always a loop, `def` always defines a function. No debate. No interpretation. They are axiomatic truths of the language.

## ⚔️ What Are Operators?

**Operators** are special symbols that perform operations on values (operands). They are the **tools of the samurai** — each with a specific purpose.

### Why Do They Exist?

Without operators, you would have to write:

```python
result = add(5, 3)
is_greater = compare_greater_than(age, 18)
both_true = logic_and(condition1, condition2)
```

Verbose, unnatural, hard to read. With operators:

```python
result = 5 + 3
is_greater = age > 18
both_true = condition1 and condition2
```

Concise, intuitive, readable. Operators are the **generative principle of code** — from simple values, you create operations, which generate new values, which create infinite complexity. **Without operators, there is no transformation. Only static values.**

---

### Operator Types

**Arithmetic operators:**

| Operator | Description | Example | Result |
|----------|-------------|---------|--------|
| `+` | Addition | `3 + 2` | `5` |
| `-` | Subtraction | `5 - 2` | `3` |
| `*` | Multiplication | `4 * 3` | `12` |
| `/` | Division | `10 / 2` | `5.0` |
| `//` | Floor division | `10 // 3` | `3` |
| `%` | Modulo | `10 % 3` | `1` |
| `**` | Exponentiation | `2 ** 3` | `8` |

**Comparison operators:**

| Operator | Meaning | Example | Result |
|----------|---------|---------|--------|
| `==` | Equal to | `5 == 5` | `True` |
| `!=` | Not equal to | `5 != 3` | `True` |
| `>` | Greater than | `10 > 5` | `True` |
| `<` | Less than | `3 < 7` | `True` |
| `>=` | Greater or equal | `5 >= 5` | `True` |
| `<=` | Less or equal | `4 <= 6` | `True` |

**Logical operators:**

| Operator | Meaning | Example | Evaluates to |
|----------|---------|---------|-------------|
| `and` | Logical "and" | `active and admin` | `True` only if both are `True` |
| `or` | Logical "or" | `x or y` | `True` if at least one is `True` |
| `not` | Negation | `not done` | Inverts the boolean value |

**Assignment operators:**

| Operator | Example | Equivalent to |
|----------|---------|---------------|
| `=` | `x = 5` | Assigns 5 to x |
| `+=` | `x += 2` | `x = x + 2` |
| `-=` | `x -= 1` | `x = x - 1` |
| `*=` | `x *= 3` | `x = x * 3` |
| `/=` | `x /= 2` | `x = x / 2` |
